## 新增机制

1. 皮革能代替兔子皮合成收纳袋
2. 白色羊毛能分解产生线
3. 鱿鱼与发光鱿鱼有40%的概率掉落黏液球
4. 附魔金苹果、三叉戟、钻石马铠与金马铠可合成
5. 在沙漠神殿考古有概率获得紫水晶
6. 部分功能方块ui材质改动
7. 末地外岛新增多种自然生成结构，丰富末地外岛的探索体验
8. 新增进度赛博科技(cybertech)，还有高手！(more_power)，这就是洪荒之力？(is_it_the_ultimate_power)，安如磐石(as_solid_as_a_rock)
9. 退出附魔台界面，放入的青金石不会弹出
10. 学徒石匠新增收购原石与深板岩的交易
11. 真实物品掉落，物品可以平躺在地面上而不会再原地旋转，右键可直接拾取物品，长按丢弃键能蓄力丢出，带有伤害词条的物品丢出后能造成伤害
12. 取消了玩家攻击与部分伤害类型的伤害冷却
13. 击杀生物有极小概率获得奇物宝藏袋，还有小概率获得随机数量硬币
14. 流浪商人售卖各种奇物
15. 用水桶收纳美西螈，获得的美西螈桶材质将能够反映美西螈的年龄与颜色
16. 猪灵堡垒桥结构的宝箱增加战利品池，必定获得熔岩漫步附魔书与远古残骸其一
17. [巡猎]附魔书可以在村庄获得
18. [毁灭]附魔书可以在林地府邸以及掠夺者前哨站集中产生
19. [存护]附魔书可以在堡垒遗迹、下界要塞、废弃传送门中获取
20. [虚无]附魔书可以在末地外岛探索宝箱获得
21. [丰饶]附魔书可以在海洋诸多宝箱中获得
22. 下界合金碎片与下界岩可合成为远古残骸
23. 命名牌可由织物、线与拴绳合成
24. 新增药水效果及相关药水物品： 
    1. 命定之死(doomed_death)：短时间内按最大生命值一定比例受到持续伤害
    2. 化身蜥蜴(evil_lizard)：大幅提升跳跃并获得八段跳能力
    3. 禁锢(durance)：无法移动，攻击速度大幅下降
25. 新增药水配方：
    1. 命定之死药水：粗制的药水+回响碎片。可以使用红石或萤石粉强化，效果相同
    2. 化身蜥蜴药水：粗制的药水+可疑的鳞片
    3. 禁锢药水：对缓慢药水使用哭泣黑曜石粉末酿造而来，可使用红石粉或萤石粉进行强化，效果相同
26. 合成地图不再需要指南针
27. 火把、灵魂火把与烈焰棒攻击敌人可使其点燃，并提高的造成的伤害
28. 参量质变仪：分为四阶质变。
     1. 普通质变消耗石英粉，支持煤炭/木炭--->萤石粉，支持对奇怪的石头进行处理
     2. 罕见质变消耗结晶青金石粉，支持紫水晶粗制物--->紫金钻。部分物品支持梭哈质变（赌博），质变后有概率复制自身，有概率直接消失，支持以下物品：不死图腾、绿宝石
     3. 稀有质变消耗哭泣黑曜石粉，支持回收铁金钻石紫水晶制的工具或装备，黏液球--->魔力黏液球
     4. 传奇质变消耗下界之星碎片，支持蔚蓝结晶--->蔚蓝锭
29. 新增村民职业
     1. 下界探险家(nether_explorer)：交易下界材料与矿物，工作方块为灵魂篝火
     2. 珠宝商(jeweler)：售卖各种矿石的商人，也会兜售奇怪的石头，工作方块为铁砧
     3. 掏垃圾桶大师(trash_can_master)：售卖垃圾袋与金色的垃圾袋，天然生成与环保屋，工作方块为垃圾桶

    1. 环保屋(green_house)：天然生成于主世界地表，同时生成一名掏垃圾桶大师

    3. 饱食(consumer)，最高3级，附魔台获得，用于剑，击杀敌人概率回复饱食度
    4. 命定之死(doomed_death)，最高5级，古代城探索获得，用于武器，命中敌人将使其获得命定之死效果

    7. 冰霜附加(ice_aspect)：最高2级，雪屋宝箱概率获得，用于近战武器，与火焰附加冲突，命中敌人将造成禁锢，移动速度与攻击速度大幅下降
    8. 金光闪闪(glimmering)：最高1级，废弃下界传送门宝箱获得，用于鞋子，与冰霜行者冲突，能使猪灵中立

    11. 逐火者(pyromania)：最高3级，下界堡垒宝箱获得，应用于近战武器，命中燃烧中的生物时造成附加伤害，与锋利、灾厄杀手、亡灵杀手、节肢杀手冲突
